# **Credit Risk Analysis – Exploratory Data Analysis (EDA)**
## **Project Overview**

- This project performs Exploratory Data Analysis (EDA) on a credit risk dataset to uncover patterns influencing loan approval and default behavior. The analysis focuses on understanding borrower characteristics, financial attributes, and historical credit information to support data-driven risk assessment.

- The project demonstrates practical data preprocessing, visualization, and analytical techniques used in real-world financial analytics workflows.

## **Objectives**

- Analyze borrower demographic and financial patterns

- Identify factors contributing to credit default risk

- Handle missing values and data inconsistencies

- Perform feature engineering for improved interpretability

- Generate insights through statistical and visual analysis

- Prepare dataset for machine learning modeling

📂 Dataset

Due to GitHub file size limitations, the dataset is hosted externally on Kaggle.

👉 Dataset Link:
🔗 https://www.kaggle.com/datasets/mishra5001/credit-card

- The dataset includes multiple tables containing applicant details, previous loan applications, and credit-related attributes used for risk analysis. It contains over 120+ features describing customer financial and application information.

## **EDA Workflow**
### **Data Cleaning**

- Missing value treatment using threshold filtering

- Column standardization

- Data type correction

### **Feature Engineering**

- Converted day-based features into years

- Binning continuous variables

- Aggregation of previous application data

### **Univariate Analysis**

- Distribution analysis

- Outlier detection

- Category frequency analysis

### **Bivariate Analysis**

- Default rate comparison across demographics

- Income vs repayment behavior

- Industry and gender risk comparison

### **Multivariate Analysis**

- Pivot tables

- Correlation heatmaps

- Feature interaction analysis

## **Key Insights**

- Income stability strongly impacts repayment behavior

- Historical credit activity influences risk prediction

- Certain demographic segments show higher default probability

- Missing data patterns provide useful analytical signals
